# Freedom Planet 2 Archipelago

An implementation of Freedom Planet 2 into the [Archipelago Multiworld Randomiser](https://archipelago.gg/), based upon the example APQuest world.

For documentation on the implementation, alongside build and installation instructions, check the [wiki page](https://github.com/Knuxfan24/Freedom-Planet-2-Archipelago/wiki). The source of the actual world can be found on my fork of the Archieplago project [here](https://github.com/Knuxfan24/Archipelago).
